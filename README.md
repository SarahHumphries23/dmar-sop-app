# DMAR SOP App (dmar-sop-app)

Simple Next.js + TypeScript starter app for a SOP library for Disaster Masters AR.
Includes:
- Searchable SOP index
- SOP detail pages with step-by-step checklist
- API routes that serve SOPs from `data/sops.json`
- Tailwind CSS for styling

## Prereqs
- Node 18+ (or Node 16+)
- npm

## Install & run locally
```bash
git clone https://github.com/SarahHumphries23/dmar-sop-app.git
cd dmar-sop-app
npm install
npm run dev
# open http://localhost:3000
